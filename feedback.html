Share Your Feedback

Your feedback drives NotesNest's evolution! Whether you have a feature suggestion, want to share your experience, or have ideas for improvements, we're listening. Every piece of feedback helps us build a better productivity app for you.

How We Use Your Feedback

Every piece of feedback is valuable to us. Here's our process:

  1. Review: We read and categorize all feedback within 24-48 hours
  2. Evaluation: We assess feasibility, impact, and alignment with NotesNest's vision
  3. Prioritization: Popular and impactful suggestions are added to our development roadmap
  4. Implementation: Selected features are developed and released in future updates
  5. Communication: We notify you when your suggestion is implemented (if you opted in)

Recent Features from User Feedback

  • Home Screen Widgets: Added in v3.0.0 - Highly requested for quick note access
  • Export Options (PDF, Text, Image): Implemented in v3.0.0 based on user needs
  • Backup & Restore: Local and cloud backup added in v3.0.0 after multiple requests
  • Dark/Light Theme Toggle: Enhanced in v2.0.0 with smooth transitions
  • Categories & Tags System: Introduced in v2.0.0 for better organization
  • Google Sign-In: Added in v2.0.0 for easier authentication
  • Drag & Drop Reordering: Implemented in v2.0.0 for note organization
  • Forget Password Feature: Added in v2.0.0 for user convenience

Feature Request Guidelines

To help us understand and evaluate your feature requests better, please consider including:

  • Use Case: Describe the problem or need this feature would solve
  • Expected Behavior: How should the feature work?
  • User Benefit: Who would benefit and how?
  • Priority: How important is this feature to your workflow?
  • Similar Apps: Have you seen this feature implemented elsewhere? How did it work?

What We're Currently Working On

Based on user feedback and our roadmap, here are some areas we're actively exploring:

  • Premium features with enhanced storage limits
  • Advanced cloud synchronization capabilities
  • Collaboration features for shared notes
  • Additional widget types and customization options
  • Enhanced tablet and landscape mode support
  • More export format options
  • Rich text formatting enhancements

Note: While we strive to implement as many suggestions as possible, not all feedback can be acted upon immediately. Factors like technical feasibility, resource availability, and alignment with NotesNest's core philosophy influence our decisions. We appreciate your understanding and patience!
